## Constraints

- No X/Twitter account (and no plans to create one)

## Highest impact channels

### Hacker News — Show HN

Lead with the Karpathy LLM Wiki angle: "LLMs can't maintain persistent knowledge across conversations — hyalo is the tooling layer that makes LLM-maintained wikis practical." HN loves Rust CLIs.

### Reddit

Each subreddit gets a different angle:

- **r/ObsidianMD** (95k+) — bulk metadata editing, link repair, CLI power-user story
- **r/ClaudeAI** — agent tooling, `hyalo init --claude`, `/hyalo-tidy` skill
- **r/LocalLLaMA** — LLM Wiki pattern, structured output for any agent
- **r/commandline** — fast Rust CLI, streaming I/O, `--jq` support
- **r/Zettelkasten** — knowledgebase maintenance at scale

### Obsidian forum & Discord

Large, active community looking for CLI tools that complement the app. The `hyalo init --claude` + `/hyalo-tidy` story is unique — no other tool has first-class agent integration.

## Medium effort, long tail

### Blog post

Write up the LLM Wiki pattern and how hyalo implements it (dev.to or personal blog). The [[research/karpathy-llm-wiki]] research note is 80% of a blog post already. This becomes the canonical link to share everywhere.

### Claude Code community

Share on Claude Code Discord / GitHub Discussions as a skill/integration example. The `init --claude` workflow is novel.

### Awesome lists

Submit to awesome-obsidian, awesome-rust, awesome-cli-apps. Free backlinks, low effort.
